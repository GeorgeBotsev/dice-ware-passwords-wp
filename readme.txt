== Description ==
This is just a simple plugin that uses the Diceware method to generate passwords for WordPress.

You can read about Diceware here: https://diceware.dmuth.org/

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Visit your profile and check the button to generate a new password and see the password suggestion.
Or
3. Add new user and see the password suggestion
